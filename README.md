# Nemchin-Taylor-Stress-Test
Цей опитувальник є комбінованою та адаптованою методикою, що поєднує підходи Т.А. Немчіна (оцінка психологічної стійкості та нервово-психічної напруги) та Дж. Тейлора (фактори маніфестної тривожності). Він призначений для визначення рівня схильності особистості до розвитку стресових реакцій та тривоги.
